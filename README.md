PROYECTO:

El proyecto se realizó en base a lo desarrollado en clase y los ejemplos colgados en el aula virtual.
- Para el ejercicio 1 se realizó un if y else para mostrar lo requerido.
- Para el ejercicio 2 se creó la lista actividades para almacenar los registros de las actividades y luego almacenarlas en el diccionario actividad.
- Para el ejercicio 3 se creó la función calcular_retorno y funciones lambda para aplicarlos a la lista creada en el ejercicio 2
- Para el ejercicio 4 se creó la clase Actividad y se hizo el ejercicio de transformar la lista que se creó anteriormente en objetos de esta clase junto a sus atributos y métodos.
